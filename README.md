# AD325 Social Media Network Project

This project implements a social media network in Python using custom
data structures and an undirected graph to manage user relationships.

**Partners:** Amy Ramsay & Wilson Yuong 
**Course:** AD325 – Data Structures  
**Due Date:** Tuesday, Dec 16

### Description
A Python implementation of a social media network that includes:
- Custom LinkedQueue and LinkedDictionary ADTs
- UserProfile and ProfileManager classes
- UndirectedGraph and Vertex for friendship connections
- BFS/DFS traversal for viewing networks
- Admin/user mode operations
- CSV profile import
- Graphviz visualization of the current user’s network
